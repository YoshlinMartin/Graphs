# Graphs
This application was an introduction to Pie and Bar graphs. It allows the user to enter in a label and a value to the charts. The first screen is the bar chart and the second is the pie chart. Both graphs are animated and have labels for each entry. Each time a label and value is entered and the user has pressed the button to add them to the graph. the textfields are cleared, ready for the next set of data and the chart displays the information in the graph. The bars in the bar graph change colour after each entry in an array that randomly selects colour and the pie chart has 5 colours that loop in an array.
